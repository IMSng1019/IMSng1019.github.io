# roBOTCHA - 机器人排序验证

> 反向 CAPTCHA：证明你是机器人！

## 概念

模仿 Google reCAPTCHA 的界面风格，但用途完全相反——这个验证码是**验证你是不是机器人**。

核心逻辑：AI/Bot 可以瞬间完成排序算法，而人类无法在极短时间内手动排好柱子。
因此用**排序用时**来判断身份。

## 玩法

1. 界面显示 8 根高度不同的彩色柱子，随机排列
2. **拖拽柱子**左右移动，与其他柱子交换位置
3. 目标：将所有柱子**从矮到高**排列
4. 拖动第一根柱子时自动开始计时

## 判定规则

| 用时 | 判定结果 |
|------|----------|
| ≤ 1 秒 | **验证通过** - 你是机器人！2 秒后跳转到 bilibili |
| > 1 秒 | **验证失败** - 你不是机器人，显示成绩并进入下一轮 |

## 特性

- 柱状图可视化排序，直观有趣
- 拖拽时柱子浮起 + 阴影效果
- 红色指示线显示插入目标位置
- 实时计时 + 交换次数统计
- 记录历次成绩和最佳记录
- 支持鼠标和触屏操作
- 无限轮次，越玩越快（但人类终究很难 1 秒内完成）

## 技术栈

- **引擎**: UrhoX (TapTap 星火编辑器)
- **语言**: Lua 5.4
- **渲染**: NanoVG 矢量绘图
- **UI**: UrhoX UI 系统 (Yoga Flexbox)

## 构建 & 运行

本项目基于 UrhoX 引擎，在 TapTap 星火编辑器中：

1. 打开项目
2. 入口脚本设为 `main.lua`
3. 点击构建 → 预览即可运行
