👋 想把你的关卡放进 Circle.html 的轮播池？

整个流程就 4 步：

  1. 左上角填好 作者名 / Pack 名，路径前缀改成 imgs/<你的用户名>/
  2. 上传图片对、拖红圈定位、加入列表（至少 2 关）
  3. 点 导出 pack JSON
  4. 把图片和 JSON 提交到对应目录，发 PR

完整步骤、目录约定、PR 模板都在投稿指南里。

🎉 导出成功！但还没结束

这个 JSON 现在只在你电脑上。要让大家能玩到，还需要：

  1. 把所有 PNG 放到 Circle/imgs/<你的用户名>/
  2. 把 JSON 放到 Circle/configs/
  3. 在 Circle/configs/manifest.json 的 packs 数组里加上你的文件名
  4. 提交 PR，描述里说一句你的整活思路
